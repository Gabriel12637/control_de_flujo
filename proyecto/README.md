# sistema de control de grass sintetico
el usuario tiene un grass el cual administra de manera manual , el usuario lo necesita que se automatize el proceso de la reserva y pago del alquiler , para cual se solicita los siguientyes casos de uso

1 el usuario podra ver las lista de horario disponible
2 el usuario podra reservar en uno de los horarios disponibles 
3 el usuario podra pagar por el alquiler de la reserva realizada
4 el usuario podra verificar su alquiler el cual mostrar los detalles 
  como la fecha, hora y costodel alquiler que realizo
